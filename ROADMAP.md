## Guidelines

- Translate meaning, not words, prefer natural Vietnamese phrasing.
- Keep all code blocks, variable names, and function names in English.
- Keep header names (`<stdio.h>`, `<math.h>`...) and all C identifiers
  in English.
- Technical terms: use the English term first, then Vietnamese in
  parentheses on first occurrence.
- No machine translation, every sentence must be reviewed by a human.
- No em dashes (`---` or `--`) in translated text; use commas or
  rewrite the sentence.
